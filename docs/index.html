真实功能

以下所有功能配置完成后开箱即用。

💬

多模型对话

流式调用 Gemini 多模态模型（文本、视觉、文档），或本机运行的任意 Ollama 模型。每会话独立切换，API Key 自己掌控。

Gemini 多模态 Ollama 流式输出
📄

文档分析

上传 Word / PDF / Excel / PPT，在对话中提问、生成摘要、提取数据表格、添加批注或转换格式。

WordPDF ExcelPPT
🗄️

知识库 RAG

从你的文件构建个人知识库。Koto 使用 FAISS 向量索引，自动将问题路由到最相关的知识源，支持多源混合检索。

FAISS 向量检索 混合检索
🧩

技能系统

用 System Prompt 定义 AI 行为，绑定快捷方式，即时切换上下文。从内置技能市场安装社区技能，或通过自然语言描述自动构建技能。

自定义提示词 技能市场 自动构建
🧠

长期记忆

Koto 持续构建用户画像，跨会话存储偏好、事实与常用上下文。记忆以 JSON 本地保存，完全掌控，无需重复自我介绍。

用户画像 本地 JSON 持久上下文
🤝

多智能体协作

Koto 将复杂请求分解给规划 Agent、执行 Agent 与技能 Agent 协同完成，共享记忆、交接子任务——远超单模型聊天机器人。

规划智能体 任务分解 结果汇总
🔁

自动化与触发器

设置定时任务——每日摘要、文件监控、定时提醒、主动问候。后台作业静默运行，不打断工作流。

定时器 主动代理 后台任务
🎙️

语音交互

点击麦克风说话，或开启 TTS 语音朗读回复。可选功能，需额外安装语音依赖。

语音输入 TTS
🖥️

快速开始

几分钟内跑起来

选择 Windows 安装包一键体验，或从源码安装以获得完整控制权。

1

下载安装包

从 GitHub Releases 下载最新的 Koto_v1.1.0_Setup.exe。

2

运行安装程序

双击运行，按向导操作完成安装。默认安装到用户目录，无需管理员权限。

3

配置 API Key

首次启动时，向导会引导你填入 API Key。免费申请 Gemini Key：aistudio.google.com

4

开始使用

Koto 以桌面窗口模式启动。立即切换模型、上传文档、构建知识库。

⬇ 下载安装包（.exe）
1

克隆并安装

git clone 仓库，创建虚拟环境，然后 pip install -r config/requirements.txt

2

填入 API Key

复制 config/gemini_config.env.example 为 gemini_config.env，填入 API Key。

3

启动 Koto

运行 Koto_Start.bat（桌面模式）或 python server.py（Web 模式，访问 localhost:5000）。

4

（可选）本地模型

安装 Ollama 并 pull 任意模型，Koto 自动识别，无需额外配置。

环境要求（源码安装）

Python 3.11+
Gemini API Key （云端模型必需）
Ollama （可选，本地模型）
Windows 10+ / macOS / Linux
